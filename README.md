# 🏥 PET 智慧排程助手 (PET Smart Scheduler)

專為核子醫學科設計的 FDG-PET/CT 智慧排程系統。  
能夠自動計算最佳打藥時間，解決 uptake time 與機器排程的衝突，確保所有病人的等待時間都在安全範圍內。

👉 **[線上試用 (Live Demo)](https://purekboy-ui.github.io/PETArr/)**

---

## ✨ 核心功能

### 1. 🤖 智慧排程演算法
系統會根據病人數量與參數，自動切換最佳模式：
- **✅ 批次模式 (Batch Mode)**：病人較少時，依序完成所有人的 Early Scan 後，再一次進行 Delayed Scan。簡單直觀。
- **🔄 自動插單模式 (Auto-Interleaved Mode)**：病人較多時，自動計算「切斷點」，在 Early Scan 之間穿插 Delayed Scan，防止後面的病人等待過久（超過 Max Wait Time）。
- **⚡ 僅 Early 模式 (Early Only)**：當設定不需要 Delayed Scan (0分鐘) 時，自動切換為連續排程模式。

### 2. ⚙️ 高度客製化參數
- **時間設定**：可調整打藥時間、掃描時間、Uptake Time。
- **限制條件**：支援設定「最小等待時間」與「最大等待時間」，確保符合臨床規範。
- **午休支援**：可設定中午空機時段（如 12:00-13:00），系統會自動避開，將排程延後。

### 3. 🎨 友善的使用者介面 (UI/UX)
- **視覺化甘特圖**：以 5 分鐘為單位的格線圖，清楚掌握機器佔用狀況。
- **清晰的視覺回饋**：不同模式有專屬的顏色與狀態顯示，Early/Delayed 任務一目瞭然。
- **響應式設計**：支援桌機、平板與手機操作。

### 4. 🖨️ 列印最佳化
- 按下 `Ctrl + P` 即可列印。
- 自動隱藏 UI 設定區塊，只保留 **「建議打藥時刻表」** 與 **「執行清單」**。
- 預留放射師簽名欄位，可直接作為正式工作表單。

---

## 🚀 如何使用

1. **設定參數**：輸入今天的病人數量，並確認掃描時間參數（預設已最佳化）。
2. **檢視排程**：系統會即時運算並顯示結果。
    - **打藥時刻表**：交給護理師，依照建議時間打藥。
    - **甘特圖**：醫師可預覽整天的機器佔用情形。
3. **執行工作**：放射師依照下方的「執行清單」依序進行掃描。
4. **列印**：需要紙本留存時，直接列印即可。

## 🛠️ 技術架構

- **Single Page Application (SPA)**：純前端架構，無後端依賴，安全性高。
- **React 18**：使用 React 進行元件化開發。
- **Deployment**：透過 GitHub Actions 自動部署至 GitHub Pages。
